# Pomodoro Timer — Modern Productivity Timer

A modern Pomodoro timer built using **HTML, CSS, and JavaScript** with a clean dark interface, smooth animations, and customizable settings.

This project demonstrates strong front-end fundamentals, UI design skills, and JavaScript logic through a fully functional productivity timer with session tracking, animated progress ring, and persistent user settings.

## How It Works

The timer follows the Pomodoro technique:

- Focus session
- Short break after each focus
- Long break after several sessions
- Optional auto-start between sessions

The progress ring updates in real time using SVG stroke animations, while user settings are saved locally so preferences remain after refresh.
